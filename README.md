## ğŸš€ How to Use

### 1. Adding Transactions
1. Select a date from the calendar
2. Choose transaction type (Income or Expense)
3. Select a category from the dropdown
4. Enter amount (supports 2 decimal places)
5. Add optional description
6. Click "Add Transaction"

### 2. Viewing Transactions
- All transactions appear in the history table
- Use filters to view specific time periods or types
- Click the trash icon to delete any transaction

### 3. Generating Reports
1. Go to "Monthly Report" section
2. Select a month from the dropdown
3. Click "Generate Report"
4. View charts, category breakdowns, and statistics
5. Click "Download as PDF" to save the report

### 4. Currency Features
- Change base currency from the header dropdown
- Use "Currency Exchange" button for live conversions
- All amounts automatically update when currency changes
